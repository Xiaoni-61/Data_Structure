Huffman编码
【题目描述】完成根据教材P128代码5.12建立Huffman编码树的源代码，包括计算各个字母对应代码的函数以及对信息进行编码与解码的函数。对于给定输入的字符串，输出Huffman编码。
对于构造的Huffman编码树，规定权值小的结点放在左子树，并且左子树的边标记为0，右子树的边标记为1。
【输入描述】一行：字符串
【输出描述】若干行：各个字母的使用频率（保留两位小数）及01的字符串编码，用英文冒号“:”隔开，输出顺序按各个字母的使用频率从大到小排列。
【样例输入】abbaaadcbd
【样例输出】
a:0.40:1
b:0.30:01
c:0.10:000
d:0.20:001
【测试案例】
【输入】delete tree delete lltll
【输出】
e:0.33:1
l:0.25:01
t:0.17:001
:0.13:0001
d:0.08:00001
r:0.04:00000
